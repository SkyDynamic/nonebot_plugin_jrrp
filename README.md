# Nonebot_Plugin_Jrrp
  
_✨ 基于OneBot适配器的[NoneBot2](https://v2.nonebot.dev/)每日人品插件 ✨_
  
</div>

## 功能

- 每天查询今日人品

## 安装

- 使用 nb-cli

```
nb plugin install nonebot_plugin_jrrp-n
```

- 使用 pip

```
pip install nonebot_plugin_jrrp-n
```

## 如何使用

# 仅限QQ群！私聊无法使用！

## 指令

使用以下指令触发

```
/jrrp(任意字符串，建议为QQ号)获取你的(任意字符串的)今日人品
/j(任意字符串，建议为QQ号)获取你的(任意字符串的)今日人品
```
